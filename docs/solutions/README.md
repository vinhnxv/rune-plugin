## How Documents Get Here

ETCHED echoes are promoted to Remembrance when they:
1. Contain a clear problem-solution pair
2. Have high confidence or 2+ session references
3. Are actionable for humans

Security category documents require `verified_by: human` before promotion.

See `plugins/rune/skills/rune-echoes/references/remembrance-schema.md` for the full schema and promotion rules.
